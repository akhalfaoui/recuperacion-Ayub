width
establece el ancho de un elemento.

max-height
se utiliza para ajustar la altura máxima de un elemento.

z-index
especifica el orden de apilamiento de un elemento.

max-width
se utiliza para establecer la anchura máxima de un elemento.

display
especifica el tipo de caja utilizada para un elemento HTML.

line-height
specifica la altura de la línea.

height
establece la altura de un elemento

overflow
especifica qué ocurre si el contenido desborda la caja de un elemento.

box-sizing
se usa para decirle al navegador lo que las propiedades de tamaño deben incluir.
