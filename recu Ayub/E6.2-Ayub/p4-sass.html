Explicación Sass
Sass es un metalenguaje de nivel superior o Cascading Style Sheets (CSS) "hojas de estilo en cascada" que se utiliza para describir el estilo de un documento de forma limpia y estructurada, con más poder que CSS. Sass proporciona sencillez, sintaxis más elegante e implementa varias características que son útiles para crear hojas de estilo manejables.

Utilización Sass
Sass permite la definición de variables. Las variables comienzan con el signo de dólar ($). La asignación de variables se hace con los dos puntos (:).
